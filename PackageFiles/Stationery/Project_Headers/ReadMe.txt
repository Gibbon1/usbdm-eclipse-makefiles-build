The USBDM New Project Wizard looks in this directory for header files to use for the 
project being created.  

This may be a single header file with the same name as the device being created 
 followed by '.h' e.g. 'MCF51CN128.h'

OR

A folder with the same name as the device being created e.g. 'MCF51CN128'.  The
 folder must contain at least the file described above.
 
The present files were generate from SVD descriptions of the devices and may be incomplete.
